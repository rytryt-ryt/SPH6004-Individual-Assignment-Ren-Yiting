# SPH6004-Individual-Assignment-Ren-Yiting

This repository contains the code and outputs for **SPH6004 (Advanced Statistical Learning) – Assignment 1 (Individual)**, based on an extracted **MIMIC-IV** ICU admission snapshot dataset. The goal is to build predictive models to estimate a patient's chance of being **discharged from ICU**, with a clear feature selection strategy and model performance comparison.

---

## Project Objectives
- Propose and implement feature selection strategies to reduce dimensionality.
- Train and compare predictive models taught in the first half of the course.
- Evaluate models using standard classification metrics and visualisations (e.g. ROC curves).
---

## Repository Structure
> File/folder names follow the workflow from data → feature selection → modelling → evaluation.

- **Data Analysis & Feature Selection/**  
  data preprocessing, implemented feature selection strategy.

- **Model: Evaluation Metrics/**  
  Metric computation scripts/utilities and summary tables.

- **Model: Logistical Regression & .../**  
  Logistic regression family models (e.g. L1/L2/Elastic Net).
  
- **Model: Tree-based Model/**  
  Tree-based models (e.g. decision tree / random forest / boosting) and evaluation.
  
- **Model: SVM/**  
  SVM training and evaluation.
  
- **Model: ROC Curves/**  
  ROC/AUC plotting scripts and exported figures.
